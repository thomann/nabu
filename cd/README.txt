*********************    NABU Offline Installation     ************
*******************************************************************

Idealfall
=========

Man öffne das File

	Nabu-***.jar

auf dieser CD durch Doppelklick. Diese Datei kann auch sonst irgendwo auf die Festplatte, einen USB-Stick oder ähnliches kopiert werden und von dort gestartet werden.


Technischer Fall
================

Man führt in einer Konsole aus:

	java -jar Nabu-***.jar

(Nabu-***.jar kann mit irgendeiner Pfadangabe wie jedes normale File geöffnet werden)

Auch in diesem Fall kann man die JAR-Datei irgendwohin kopieren.


Sonst bzw. bei Fehlfunktionen
=====      ==================

Ein Java muss installiert sein (wohl mindestens Version 1.5 [=5.0]). Unter MacOS X ist dies der Fall. Unter Windows und *n*ix nicht unbedingt.

** Windows **
Ich habe einige aktuelle (Mai 2006) Versionen heruntergeladen. Unter Windows reicht im Prinzip das

	jre-1_5_0_06-windows-i586-p.exe

Zur Sicherheit (falls es so nicht klappt) habe ich auch noch das jdk (Development Kit - eigentlich zum weiterentwickeln) mitgepackt.
